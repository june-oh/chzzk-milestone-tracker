# 📊 소프트콘(Softcon) 및 커뮤니티 데이터 활용 과거 마일스톤 날짜 추적 가이드

본 가이드는 첫 수집 이전에 달성된 각 치지직 스트리머들의 **과거 1,000시간 단위 마일스톤(1,000H, 2,000H 등) 돌파 일자**를 외부 커뮤니티 및 통계 사이트에서 수색하여 대시보드에 완벽히 채워 넣는 프로세스를 설명합니다.

---

## 🔍 Step 1. 마일스톤 일자 추적하기

### 방법 A. 소프트콘 뷰어쉽 (Softcon Viewership) 활용하기 (가장 정확)
소프트콘 뷰어쉽은 대한민국 트위치/치지직/아프리카TV 크리에이터들의 누적 방송 시간 및 통계를 장기 보관하고 있는 통계 전문 사이트입니다.

1. **소프트콘 뷰어쉽 공식 웹사이트**([https://viewership.softcone.net/](https://viewership.softcone.net/))에 접속합니다.
2. 상단 검색창에 타겟 스트리머의 닉네임(예: `초승달`, `아리사` 등)을 입력하여 채널 전용 통계 페이지로 진입합니다.
3. **[방송 기록 / 통계]** 탭 또는 **[상세 통계 추이]** 차트 항목을 확인합니다.
4. 차트의 X축(날짜)과 Y축(누적 방송 시간)을 스크롤하면서, **누적 방송 시간이 `1000`, `2000`, `3000` 등을 최초로 초과하여 도달한 일자**를 메모합니다.
   - 예: 초승달님의 누적 방송 수치가 `4,000시간`을 처음으로 넘어선 날짜가 차트상 `2025-04-15`로 확인된다면, 해당 날짜가 4,000H 마일스톤 달성일입니다.

### 방법 B. 치지직 네이버 팬카페 및 라운지 검색하기
팬들이 남긴 축하 게시글이나 공지글을 통해 초정밀 날짜를 얻을 수 있습니다.

1. 스트리머의 네이버 팬카페 혹은 치지직 공식 라운지에 접속합니다.
2. 게시글 검색창에 다음 키워드로 검색을 수행합니다:
   - `1000시간`, `2000시간`, `3000시간`, `천시간`, `달성`
   - `방송 시간`, `마일스톤`
3. "오늘 방송 시간 2000시간 돌파 축하드려요!", "[공지] 방송 누적 3000시간 달성 기념 Q&A" 와 같이 팬들이 올린 **게시글의 작성 일자(연월일)**를 채택합니다.

---

## 🛠️ Step 2. 데이터베이스(Vercel KV)에 저장하기

수집된 데이터를 대시보드에 반영하는 작업은 동봉된 **`fill_history.js`** 작동 스크립트를 통해 손쉽게 처리할 수 있습니다.

### 1. `fill_history.js` 파일 열기
프로젝트 루트 디렉토리에 생성된 `fill_history.js` 파일을 텍스트 에디터로 엽니다.

### 2. 수집된 데이터 입력하기
파일 내부의 `const HISTORICAL_DATA` 구역에 찾으신 채널 ID, 스트리머 이름, 마일스톤 시간, 날짜를 다음과 같이 채워 넣습니다:

```javascript
const HISTORICAL_DATA = [
  {
    channelId: "65c3035bdc598c81f15a8fe0e958b3ce", // 초승달
    channelName: "초승달",
    milestone: 4000,
    date: "2025-04-15T12:00:00Z" // 수집한 날짜 (ISO-8601 형식 권장)
  },
  {
    channelId: "4de764d9dad3b25602284be6db3ac647", // 아리사
    channelName: "아리사",
    milestone: 7000,
    date: "2025-05-10T12:00:00Z"
  }
];
```

### 3. 스크립트 실행하여 DB에 주입하기
터미널에서 아래 명령을 실행합니다. Vercel KV 연결을 위해 로컬 환경변수(`.env.local` 또는 `.env`)가 로드된 상태여야 합니다.

```bash
# dotenv-cli 가 설치되어 있는 경우 (강력 권장)
npx dotenv-cli -- node fill_history.js

# 또는 일반 node 실행 (로컬에 환경변수 세팅 완료된 상태)
node fill_history.js
```

## ✨ Step 3. 대시보드에서 확인

스크립트가 성공적으로 수행되면 대시보드는 다음과 같이 변경됩니다:

1. **명예의 전당 (메인 화면 하단)**: 새로 주입한 과거 마일스톤 로그가 실제 달성 일자와 함께 최신순 리스트에 자동으로 등록됩니다.
2. **스트리머 상세 페이지**:
   - 기존의 `데이터 없음 (추정)`으로 흐리게 표기되던 과거 마일스톤 일지가 **수집된 정확한 실제 돌파 날짜**로 대체됩니다.
   - 우측 상태 뱃지가 `추정치`에서 초록색의 `정상 인증 완료` 뱃지로 실시간 변경됩니다.
   - 성장 곡선 그래프상에서도 해당 날짜 좌표에 마일스톤 노드 점이 정확히 위치하게 됩니다.
